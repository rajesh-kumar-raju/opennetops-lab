OpenNetOps Labs

OpenNetOps Labs focuses on SONiC/open networking adoption, AI-assisted observability, telemetry integration, and network automation.

Focus Areas

  • SONiC PoC and deployment readiness
  • Telemetry integration using Prometheus, Grafana, and Loki
  • AI-assisted root cause analysis for network issues
  • Packet visibility and observability workflows
  • Data-center network automation
  • BGP, EVPN/VXLAN, and fabric health monitoring

Business Value

We help enterprises, service providers, and infrastructure teams evaluate and operationalize open networking platforms by building practical PoCs, observability dashboards, and AI-driven troubleshooting workflows.
